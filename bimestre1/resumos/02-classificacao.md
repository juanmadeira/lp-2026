# RESUMO - Categorias e classificação das linguagens de programação
## Classificação por nível
Indicativo da CAPACIDADE DE ABSTRAÇÃO.

### Nível baixo:
- Manipulação da memória
- Pouca ou nenhuma abstração do hardware
- Acesso aos registradores
- Geralmente instruções têm correspondência com a frequência do relógio (clock cycle)
- Excelente desempenho

Exemplos:
- Assembly
- Assembler (compilador)
- Linguagem de máquina

### Nível médio
- Manipulação de memória
- Presença de abstração do hardware junto de comandos para manipulá-lo
- Alto nível de abstração (funções, instruções)
- Bom desempenho

Exemplo:
- C

### Nível alto
- Independentes da arquitetura
- Abstração sobre o funcionamento da máquina
- Altos níveis de abstração no código (funções, instruções e objetos)
- Maior facilidade na escrita e legibilidade do código
- Mais populares

Exemplos:
- Java
- C++
- Prolog
- Python
- Scala
- Ruby

## Classificação por geração
### 1ª geração
Linguagem de máquina ou binária, não necessita de qualquer tradutor ou compilador para ser executada.

### 2ª geração
Uso de notação simbólica, códigos binários são substituídos por mnemônicos.

Exemplo:
- Assembly

### 3ª geração
Introdução do uso de instruções de entrada/saída, cálculos lógicos ou aritméticos e controle de flexo. Introdução de variáveis e funções. Requerem um tradutor. Independentes do hardware.

Exemplos:
- C
- Java
- Python

### 4ª geração
Linguagens que consistem em declarações similares à linguagem humana, usadas principalmente em programação de base dados e scripts.

Exemplos:
- SQL
- Oracle
- MATLAB

### 5ª geração
Empregadas na área de inteligência artificial, facilitam representar o conhecimento da inferência de dados a partir de informação codificada.

Exemplos:
- Prolog
- Lisp
